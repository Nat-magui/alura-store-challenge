# 🏪 Alura Store Challenge – Análisis de Datos

## 📌 Descripción del proyecto

Este proyecto forma parte del **Challenge de Data Science de Alura Latam**.
El objetivo del análisis es ayudar al **Sr. Juan**, propietario de la cadena **Alura Store**, a decidir cuál de sus cuatro tiendas debería vender para invertir en un nuevo emprendimiento.

Para tomar esta decisión, se analizaron datos de ventas de las cuatro tiendas, evaluando distintos factores como ingresos, categorías de productos, valoraciones de clientes, productos más vendidos y costos de envío.

El análisis se realizó utilizando **Python, Pandas y Matplotlib** en **Google Colab**.

---

# 🎯 Objetivos del análisis

El objetivo principal es identificar **qué tienda presenta el menor rendimiento** considerando:

* Ingresos totales por tienda
* Categorías de productos más vendidas
* Valoraciones promedio de los clientes
* Productos más y menos vendidos
* Costo promedio de envío

A partir de estos datos se generan visualizaciones y conclusiones que permiten recomendar qué tienda debería vender el Sr. Juan.

# 📊 Análisis realizados

Durante el proyecto se realizaron los siguientes análisis:

### 1️⃣ Ingresos totales por tienda

Se calcularon los ingresos totales sumando el valor de las ventas de cada tienda.

### 2️⃣ Categorías de productos más vendidas

Se analizaron las categorías con mayor volumen de ventas para entender las preferencias de los clientes.

### 3️⃣ Valoraciones promedio de los clientes

Se evaluó el nivel de satisfacción de los clientes utilizando la calificación promedio de las compras.

### 4️⃣ Productos más y menos vendidos

Se identificaron los productos con mayor y menor demanda dentro de cada tienda.

### 5️⃣ Costo promedio de envío

Se calculó el costo promedio de envío para analizar el impacto logístico en cada tienda.

* Gráficos de dispersión para explorar relaciones entre variables

# 🧠 Conclusión

Tras analizar los ingresos, las categorías de productos, las valoraciones de los clientes, los productos más vendidos y los costos de envío, se observó que **la Tienda 4 presenta el menor ingreso total entre las cuatro tiendas**.

Aunque sus valoraciones y categorías vendidas son similares a las demás, su rendimiento financiero es inferior.

Por esta razón, se recomienda que **el Sr. Juan considere vender la Tienda 4**, ya que representa la tienda con menor desempeño dentro del conjunto analizado.
